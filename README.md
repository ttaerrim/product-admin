### `Installation`

```
# clone the project
$ git clone https://github.com/ttaerrim/product-admin.git

# install modules
$ cd product-admin
$ npm install || yarn

# start
$ npm start || yarn start

⠀
⠀  You can now view this project in the browser.
⠀  http://localhost:3000/
⠀
```
